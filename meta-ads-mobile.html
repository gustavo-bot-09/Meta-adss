Conecte sua conta Meta Ads
Puxe métricas reais automaticamente: investimento, receita, ROAS, campanhas, criativos — direto do Gerenciador de Anúncios, sem precisar inserir dados manualmente.
1
Criar App no Meta Developers
Acesse developers.facebook.com → Meus Apps → Criar App → Tipo: Negócios. Anote o App ID.
2
Gerar Access Token no Explorer
Acesse developers.facebook.com/tools/explorer → selecione seu App → adicione permissões ads_read e ads_management → clique em Gerar Token de Acesso.
3
Cole o token abaixo e conecte
Tokens do Explorer duram ~1h. Para uso prolongado, troque por um token de longa duração (60 dias) via endpoint oauth/access_token.
💡 Privacidade: O token é salvo apenas no seu dispositivo (localStorage). Nunca trafega para servidores externos.

📲 Instalar como App
Adicionar à tela inicial do celular

Instale o dashboard como app nativo — funciona sem barra do navegador, igual a um app da loja.

No Safari, siga os passos:

1️⃣Toque em Compartilhar ⬆ na barra inferior do Safari
2️⃣Role e toque em "Adicionar à Tela de Início"
3️⃣Confirme o nome "Meta Ads" e toque em Adicionar
✅
App instalado!
Meta Ads está na sua tela inicial.
⚠ Arquivo local detectado — o Chrome exige HTTPS para instalar PWA. Escolha uma das opções abaixo.
Opção 1 — Mais fácil
1️⃣ Acesse github.com no celular → crie conta gratuita (ou faça login)
2️⃣ Crie um repositório público → clique em Add file → Upload files → envie o meta-ads-mobile.html → renomeie para index.html
3️⃣ Vá em Settings → Pages → Branch: main → Save. Aguarde ~1 min.
4️⃣ Abra a URL seu-usuario.github.io/repositorio no Chrome → ícone ⊕ aparece na barra → instale!
Opção 2 — Via PC
1️⃣ No PC, abra o terminal na pasta do arquivo e rode: npx serve . (precisa do Node.js) — ou use o VS Code com a extensão Live Server
2️⃣ Conecte o celular na mesma rede Wi-Fi → acesse http://IP-do-PC:3000 no Chrome
3️⃣ No Chrome: ⋮ → Adicionar à tela inicial → o app instala normalmente

Abra este arquivo no Chrome (Android) ou Safari (iPhone) para instalar.

💡 No Chrome: procure o ícone ⊕ na barra de endereço ou ⋮ → Adicionar à tela inicial.
